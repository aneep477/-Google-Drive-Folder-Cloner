# 🚀 Google Drive Folder Cloner 

Sistem berasaskan web untuk menyalin keseluruhan folder Google Drive ke folder lain dengan satu klik, lengkap dengan **Progress Bar** dan **Struktur Folder Teratur**.

### 🌟 Ciri-Ciri Utama
- ✅ **Salin Folder Induk:** Tidak bersepah, folder dicipta secara automatik.
- ✅ **Progress Bar:** Lihat perkembangan proses secara real-time.
- ✅ **Safety Lock:** Amaran pop-up jika cuba menutup tab semasa menyalin.
- ✅ **UX Menarik:** Antaramuka bersih menggunakan Bootstrap 5.

### 🛠️ Cara Penggunaan
1. **Klik Butang di Bawah** untuk membuka aplikasi:
   [![Open Web App](https://img.shields.io/badge/Launch-Web%20App-blue?style=for-the-badge&logo=google-chrome)](MASUKKAN_URL_WEB_APP_ANDA_DI_SINI)

2. Masukkan **URL Folder Sumber** yang ingin dicopy.
3. Masukkan **URL Folder Destinasi** anda.
4. Klik **Mula Salin** dan tunggu sehingga pop-up "Berjaya" muncul.

---
*Dibina menggunakan Google Apps Script & Bootstrap.*
